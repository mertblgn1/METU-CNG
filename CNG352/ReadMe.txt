Before running the app, make sure you have the following installed:

- Flutter: Install Flutter.
- MySQL: Install MySQL.
- Flask: If you're using the backend server, install Flask to run the server.

## Setting Up the Backend

1. Create a MySQL database for the app by using dbCreation.sql or cocktail_db_creation.sql.
2. Run the provided SQL script to set up the necessary tables and data.
3. Start the Flask server by running the command: flask run.

Alternatively, you can use an IDE like PyCharm to run the Flask app.

## Running the App

- Open the project by using Intellij.
- Ensure your Android emulator or iOS simulator is running, or connect a physical device.
- Run the app using the following command: flutter run.
